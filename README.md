# tic-tac-toe

A basic tic-tac-toe application written in Rust, used as a learning exercise in Rust and UI development.

The TUI is created using [tuirealm](https://github.com/veeso/tui-realm/).

> [!NOTE]
> The TUI is not intended to be a showcase of design or usability, but rather a sandbox for learning how to build TUIs in Rust. It is intentionally simple and unpolished, with known limitations for small terminal sizes.
